# Utilities class

Fancy.css provides some utilities class you can use in any place you needed.

Any utilitie class begins with `u-` in the name.

`u-margin-bottom-bg`

## Margin

- `u-margin-bottom-sm`
- `u-margin-bottom-md`
- `u-margin-bottom-bg`
- `u-margin-top-sm`
- `u-margin-top-md`
- `u-margin-top-bg`
- `u-margin-horizontal-auto`

## Padding

- `u-padding-bottom-sm`
- `u-padding-bottom-md`
- `u-padding-bottom-bg`
- `u-padding-top-sm`
- `u-padding-top-md`
- `u-padding-top-bg`
- `u-padding-horizontal-auto`
